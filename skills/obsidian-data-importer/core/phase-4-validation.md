# Phase 4: Validation & Preview

## Purpose

Verify template correctness and generate sample output.

## Process

1. Validate YAML syntax
2. Check Handlebars syntax
3. Apply template to first data row
4. Detect potential issues
5. Generate validation report

## Outputs

- Validation status (pass/fail)
- Preview of first note
- List of warnings or issues
- Recommendations for fixes
