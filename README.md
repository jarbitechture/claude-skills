# Claude Code Skills Collection

1,253 skills for [Claude Code](https://claude.com/claude-code) covering automation, development, security, infrastructure, and more.

## Structure

Each skill lives in its own directory under `skills/` and typically contains:
- `SKILL.md` — The skill prompt and instructions
- `references/` — Supporting documentation and examples (when applicable)

## Categories

- **Automation** — SaaS integrations (Slack, Notion, GitHub, Salesforce, HubSpot, etc.)
- **Development** — Code refactoring, test generation, API design, frontend/backend patterns
- **Security** — Vulnerability scanning (Semgrep, CodeQL), threat modeling, security best practices
- **Infrastructure** — Terraform/IaC, Kubernetes troubleshooting, CI/CD optimization, Docker
- **Data** — Database optimization, CSV analysis, data pipelines, knowledge graphs
- **AI/ML** — Hugging Face tools, model training, evaluation, DSPy, agent frameworks
- **Architecture** — ADRs, software architecture, technical debt management
- **Research** — Web research, paper analysis, codebase exploration

## Installation

Copy any skill directory into `~/.claude/skills/` to make it available in Claude Code.
